TL;DR
- SOW с блоками предоплаты и чётким DoD, чтобы защитить интересы и избежать хаоса.

Commercials
- Rate: $40/h (текущая договорённость) — цель вернуть к $50/h при росте объёма.
- Block 1 (Discovery): 40h = $1,600 — предоплата 100%.
- Block 2 (MVP Sprint 1): 80h = $3,200 — предоплата 100%.
- Blocks 3..N: по 80h, предоплата 100%.
- Invoices: USDT TRC‑20.

Scope (Discovery → MVP)
- Discovery (40h): C4 контекст/контейнеры, события и API контракты, ledger выбор, банк DvP shortlist, KEP/УКЭП вендор.
- MVP (3–4 спринта): Registry, Identity/KYC, Tokenization, Settlement DvP, Custody, Disclosure.

DoD (per slice)
- API контракты + события + тест‑кейсы.
- Логи аудита и хеш‑якорь.
- Документация (README per service).

Governance
- Weekly demo, scope freeze per sprint, change requests через доп. блоки.
- Single comms channel, SLA ответов 24h.
